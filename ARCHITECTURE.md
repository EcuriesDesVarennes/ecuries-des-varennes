## Images

Each page has its own `images/` folder:

- `home/images/`
- `nos-activites/images/`
- `le-domaine/images/`
- `galerie/images/`
- `contact/images/`

Place editorial photos in numbered files:

- `1.jpg`
- `2.jpg`
- `3.jpg`
- `4.jpg`
- `5.jpg`

Use the numbers in order. `1` is the first image shown, then `2`, and so on.

## Notes

- The root `index.html` is only a redirect to `home/` for GitHub Pages compatibility.
- Shared logos stay in `/img` for now because they are reused site-wide.
